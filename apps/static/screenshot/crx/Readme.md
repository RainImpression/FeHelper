Web前端助手（FeHelper）- crx文件安装方法
=============================

### 安装方法 
1. 下载当前最新版`*.crx`
1. chrome浏览器地址栏输入：[chrome://extensions/](chrome://extensions/) 并打开
2. 右上角开启`开发者模式`
3. 拖拽crx到当前页面，完成安装
